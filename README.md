# job-ai

Страница тарифов: калькулятор периода доступа, блоки стоимости, сайдбар.

## Стек

- React 19, TypeScript
- Ant Design (Select, Tooltip)
- SCSS, CSS Modules
- ESLint, Prettier, Stylelint

Структура кода близка к Feature-Sliced Design: `app`, `pages`, `widgets`, `features`, `entities`, `shared`.

## Запуск

```bash
npm install
npm start
```

Откроется dev-сервер: http://localhost:3000

## Сборка

```bash
npm run build
```

Результат в папке `build`.
